# 💰 Expense Tracker App

A clean, responsive expense tracking application built with vanilla HTML, CSS, and JavaScript — no frameworks or libraries needed.

## 📸 Preview

> Open `index.html` in your browser to see it live.

## ✨ Features

- ➕ Add income and expense transactions
- 🔢 Automatic balance, income & expense calculation
- 🟢 Green / 🔴 Red color feedback for positive/negative balance
- 📋 Full transaction history with formatted amounts
- 🗑️ Delete individual transactions
- 💾 LocalStorage support — data persists after page refresh
- 📭 Empty state message when no transactions exist
- ⚠️ Input validation with error messages

## 🛠️ Technologies

| Technology | Purpose |
|------------|---------|
| HTML5      | Page structure & markup |
| CSS3       | Styling, layout, animations |
| JavaScript | Logic, DOM manipulation, localStorage |

## 🚀 Getting Started

No installs or setup required.

1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Start tracking your expenses!

```bash
git clone https://github.com/regis-chifambaa/expense-tracker.git
cd expense-tracker
open index.html
```

## 💡 How to Use

| Action | How |
|--------|-----|
| Add income | Enter description + positive amount (e.g. `500`) |
| Add expense | Enter description + negative amount (e.g. `-30`) |
| Delete a transaction | Click the **✕** button on any transaction |
